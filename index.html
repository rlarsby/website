Master's Thesis

P4

Personal Rapid Transit Visualization

This project used CAD to design a personal rapid transit (PRT). The PRT was imported into Unreal Engine 4 where the creation of an environment took place which visualized the user expereience of the PRT. During the project Adobe Illustrator, Fusion 360, Unreal Engine 4, OBX, Resolve and Blender were used to reach the end-result.

A Luxurious Teacup

The purpose of this project was to come up with an idea of a product, that normally does not contain wooden materials, and combine wood and some other material in the product.
